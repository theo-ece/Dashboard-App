# Lab 4 Middleware and Unit testing

## Introduction

The basic Node.js application on TypeScript.

## Installing

```bash
git clone https://github.com/Jedhug/Dashboard-App.git
cd Dashboard-App
npm install
npm install aes-js
```

## Build

```bash
npm run build
```

## Development

```bash
npm run dev
```

## Database population

```bash
npm run populate
```
